## Key Features

- **Free PDF Book Library**: Browse, read, and download books in PDF format without any cost.
- **No Subscription Required**: Completely free to use with no hidden charges or premium plans.
- **Wide Range of Categories**: Academic books, fiction, non-fiction, technology, business, and more.
- **Clean User Interface**: Easy-to-navigate interface with search functionality for quick access to your desired books.
- **Download Option**: Download books for offline reading in PDF format.
- **Cloud-Hosted**: Books are stored securely and served via Cloudinary for fast, reliable access.

---

## How it Works

1. **Browse the Library**: Explore thousands of available books by category or search by title and author.
2. **Read Online**: Open any book in your browser instantly without needing to download.
3. **Download PDF**: Save the book to your device in PDF format for offline reading.
4. **Completely Free**: No login required, no subscription fees, just free knowledge.

---

## Technologies Behind Readora

- **Next.js**: Full-stack framework for fast and scalable web applications.
- **Prisma & NeonDB**: Serverless relational database managing book records.
- **Tailwind CSS**: Modern utility-first framework for beautiful, responsive UI design.
- **Cloudinary**: For hosting book cover images and PDF files efficiently.
- **React Query & React Hook Form**: Seamless data fetching and form handling.
- **Shadcn/ui**: Reusable, accessible components to ensure a polished user experience.
