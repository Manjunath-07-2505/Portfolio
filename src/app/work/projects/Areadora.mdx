## Overview

The **Bank Management System** is a Java-based console application developed using **JDBC (Java Database Connectivity)** to perform essential banking operations.  
It allows users to manage bank accounts efficiently while ensuring secure interaction with a **MySQL database**.

This project focuses on implementing real-world banking functionalities such as account creation, deposits, withdrawals, and balance inquiry, while demonstrating strong backend concepts like database connectivity, exception handling, and the **DAO design pattern**.
