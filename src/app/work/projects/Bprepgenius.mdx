## Overview

**MealMate** is a full-stack web application built using the **MERN stack (MongoDB, Express, React, Node.js)** that helps users manage meals and recipes efficiently.  
The platform allows users to explore food items, upload images, and interact with a responsive frontend connected to a robust backend API.

This project demonstrates real-world full-stack development concepts including REST APIs, database modeling, authentication-ready architecture, and clean frontend-backend separation.
