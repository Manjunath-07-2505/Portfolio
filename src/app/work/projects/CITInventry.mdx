## Overview

**Planify** is a clean and user-friendly **task management (To-Do) application** developed using **React and Vite**.  
It allows users to efficiently add, track, complete, undo, and delete tasks while categorizing them by **priority** and **type**.

The application focuses on simplicity, usability, and performance, providing a smooth experience with support for **Light Mode and Dark Mode**.

- **Progress Indicator**: Displays the number of completed tasks visually.
- **Light & Dark Mode**: Toggle between themes for better user comfort.
- **Responsive UI**: Works smoothly across different screen sizes.
- **Fast Performance**: Built using Vite for rapid development and optimized builds.

## âš™ï¸ How It Works

1. Users enter a task in the input field.
2. Tasks can be assigned a **priority level** (Low / Medium / High) and category.
3. Tasks are displayed in a list with action buttons.
4. Users can:
   - Mark tasks as completed
   - Undo completed tasks
   - Delete tasks
5. The progress bar updates dynamically based on task completion.
6. Theme toggle allows switching between Light and Dark mode.
